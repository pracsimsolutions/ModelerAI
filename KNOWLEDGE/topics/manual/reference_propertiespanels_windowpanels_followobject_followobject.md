# Follow Object

The Follow Object panel allows you to set the 3D view to follow an object.

 

 

The following properties are on the Follow Object panel:

 

### Object

 

Set or clear the object to follow.

 

### Rotate with Object

 

When checked, the view will rotate with the object.
