# Composite Staytime by Hour vs Time

## Overview and Key Concepts

 

 

This composite template creates a line graph. The line shows the average staytime for the previous hour, every hour.

 

For more information about Staytime templates, see Staytime Templates.

 

 

 

## Properties Panels

 

The Composite Staytime by Hour vs Time template uses the following properties panels:

 

- Options
- Settings
- Text
- Colors
- Sorting
- Advanced
