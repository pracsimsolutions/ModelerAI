# Text

The Text panel defines what text is displayed and how large it is.

 

 

The following properties are on the Text panel:

 

### Text Display

 

The text that will be displayed by the Text object. This can be a dynamic value like the current model time.

 

### Text Size

 

The size of the text object.

 

### Thickness

 

The thickness of the text object. Increasing this above zero will change the text object from a 2D display into a 3D text object.
