# Staytime by Type Bar Template

## Overview and Key Concepts

 

 

This template creates a bar chart. Each unique Type value included in the template is represented by a bar. The length of that bar shows the Average Staytime for items of that type.

 

For more information about Staytime templates, see Staytime Templates.

 

 

 

## Properties Panels

 

The Staytime by Type Bar template uses the following properties panels:

 

- Options
- Settings
- Text
- Colors
- Sorting
- Advanced
