# Composite State Bar Template

## Overview and Key Concepts

 

 

This composite template creates a bar chart, and calculates utilization based on each object's time in various states. Each segment of the bar chart represents the average proportion of time spent in one of the object's states. The utilization shown is the average utilization of all objects.

 

 

 

## Properties Panels

 

The Composite State Bar template uses the following properties panels:

 

- Options
- Settings
- Text
- Sorting
- Advanced
