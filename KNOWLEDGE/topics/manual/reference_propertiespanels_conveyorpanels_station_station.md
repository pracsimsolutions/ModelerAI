# Station

The Station panel defines properties associated with processing an item on the station.

 

 

The following properties are on the Station panel:

 

### Process Condition

 

Returns a value, 1 or 0, defining the condition by which a given item passing over the station should be processed. Return 1 to process the item. Return 0 to let the item pass without processing it.

 

### Process Time

 

Defines the processing time for a given item at the station.

 

### Use Operator

 

Defines settings for calling an operator to the station to process the item. Check the box and then define the reference to the operator or dispatcher to give the task to, as well as the # of Operators to request, and the priority and preemption values of the request task sequence.
