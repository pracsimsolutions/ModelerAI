# Object Creation

The Object Creation panel allows you create new objects similar to other existing objects.

 

 

The following properties are on the Object Creation panel:

 

### Use Same Properties As

 

Choose the Object you want to copy properties from when creating a new object. This object must be of the same class as the one you're creating. You can either sample an object in the model or you can pick from a list of recently chosen objects.

 

### Create as Template Instance

 

When checked, created object will be a template instance of the reference object chosen in the field above instead of just copying the values of that object's properties.
