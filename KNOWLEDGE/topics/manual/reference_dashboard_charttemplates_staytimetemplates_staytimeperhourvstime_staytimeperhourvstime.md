# Staytime by Hour vs Time

## Overview and Key Concepts

 

 

This template creates a line graph. Each object is connected with a unique line. Each line shows the average staytime for the previous hour, every hour, for that object.

 

For more information about Staytime templates, see Staytime Templates.

 

 

 

## Properties Panels

 

The Staytime by Hour vs Time template uses the following properties panels:

 

- Options
- Settings
- Text
- Colors
- Sorting
- Advanced
