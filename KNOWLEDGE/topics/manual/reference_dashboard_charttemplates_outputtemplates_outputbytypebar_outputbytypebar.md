# Output by Type Bar Template

## Overview and Key Concepts

 

 

This composite template creates a bar chart. Each unique type is represented by a bar. The length of that bar is determined by the total throughput of that type. For more information about composite Output templates, see Output Templates.

 

 

 

## Properties Panels

 

The Output by Type Bar template uses the following properties panels:

 

- Options
- Settings
- Text
- Colors
- Sorting
- Advanced
