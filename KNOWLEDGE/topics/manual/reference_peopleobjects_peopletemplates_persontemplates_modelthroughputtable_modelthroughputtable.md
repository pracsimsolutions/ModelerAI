# Model Throughput Table People Template

## Overview and Key Concepts

 

 

This template calculates and displays total number of flowitems (created with the Create Person activity) that have exited the model using a Remove Person activity.

 

For more information about People Templates, see the People Templates topic.

 

### Subset and Partition Limits

 

This template does not limit subset or partitions.

 

 

 

## Properties Panels

 

This template uses the following properties panels:

 

- Data
- Subsets
- Partitions
- Text
- Advanced
