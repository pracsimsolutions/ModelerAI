# Process Flow Variables

The Process Flow Variables panel displays a list of the flow's Process Flow Variables.

 

 

The following properties are on the Process Flow Variables panel:

 

### Process Flow Variables

 

The list of process flow variables defined for the flow. If this panel is showing on a flow's instance object then this list will only show user accessible variables.
