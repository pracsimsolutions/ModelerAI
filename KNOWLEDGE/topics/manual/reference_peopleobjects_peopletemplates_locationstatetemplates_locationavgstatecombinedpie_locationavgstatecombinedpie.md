# Location Avg State (Combined) Pie Template

## Overview and Key Concepts

 

 

This chart template creates a pie chart, and calculates utilization based on each object's time in various states.

 

 

 

## Properties Panels

 

This template uses the following properties panels:

 

- Data
- Options
- Subsets
- Partitions
- Settings
- Text
- Colors
- Sorting
- Advanced
