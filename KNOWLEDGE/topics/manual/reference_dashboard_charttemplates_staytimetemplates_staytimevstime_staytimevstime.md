# Staytime vs Time Template

## Overview and Key Concepts

 

 

This template creates a time plot. Each point on the plot corresponds with an item exiting one of the included objects. The horizontal position indicates when the item exited, and the vertical position indicates how long the object had stayed in the object. Each object is represented by a different color.

 

For more information about Staytime templates, see Staytime Templates.
