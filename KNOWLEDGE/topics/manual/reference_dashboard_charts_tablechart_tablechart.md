# Table Chart

## Overview and Key Concepts

 

The table chart can be used to display the data in a Statistics Collector table or Calculated Table as a chart.

 

 

 

 

## Performance Measures

 

The Table chart provides aggregates of each included numeric column as performance measures. For each column, the chart provides the Average, Sum, Minimum, and Maximum as a possible performance measure.

 

 

 

## Properties Panels

 

The Table Chart uses the following properties panels:

 

- Data
- Text
