# Utilization By Hour Template

## Overview and Key Concepts

 

 

This chart template creates a time plot. At the end of each hour (interval), it calculates the percentage of time in which the object(s) spent utilized and plots the result as a percentage. Each value is plotted consecutively throughout the run time of the model as a stair step by default.

 

 

 

## Properties Panels

 

The Utilization By Hour template uses the following properties panels:

 

- Options
- Settings
- Text
- Colors
- Sorting
- Advanced
