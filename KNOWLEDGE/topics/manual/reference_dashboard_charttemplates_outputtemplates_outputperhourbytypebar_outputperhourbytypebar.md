# Output per Hour by Type Bar Template

## Overview and Key Concepts

 

 

This composite template creates a bar chart. Each unique type is represented by a bar. The length of that bar is determined by the average throughput of that type per hour. For more information about composite Output templates, see Output Templates.

 

 

 

## Properties Panels

 

The Output per Hour by Type Bar template uses the following properties panels:

 

- Options
- Settings
- Text
- Colors
- Sorting
- Advanced
