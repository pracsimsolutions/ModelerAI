# Texture

The Texture panel defines what texture is draw on the plane.

 

 

The following properties are on the Texture panel:

 

### File Path

 

The file path to a 2D image used to texture the plane.

 

### X Repeat

 

The number of times the texture is repeated in the horizontal direction.

 

### Y Repeat

 

The number of times the texture is repeated in the vertical direction.

 

### Divisions

 

The number of sides to be drawn on a column or sphere shape.
