# Tracked Variable Bar Template

## Overview and Key Concepts

 

 <!-- This section explains what data the chart shows, and any assumptions it makes --> 

This template creates a bar chart. Each specified tracked variable is represented by a bar group. The length of the bar is determined by the value of the statistic for that tracked variable.

 

For more information about Tracked Variable templates, see Tracked Variable Templates.

 

 

 

## Properties Panels

 

The Tracked Variable Bar template uses the following properties panels:

 

- Options
- Settings
- Text
- Colors
- Sorting
- Advanced
