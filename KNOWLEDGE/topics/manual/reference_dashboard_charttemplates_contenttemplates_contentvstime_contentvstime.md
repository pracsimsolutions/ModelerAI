# Content vs Time Template

## Overview and Key Concepts

 

 

This template creates a stair-step chart. Each specified object is represented by a different color, which shows current content (or WIP) over time.

 

For more information about Content templates, see Content Templates.

 

 

 

## Properties Panels

 

The Content vs Time template uses the following properties panels:

 

- Options
- Settings
- Text
- Colors
- Sorting
- Advanced
