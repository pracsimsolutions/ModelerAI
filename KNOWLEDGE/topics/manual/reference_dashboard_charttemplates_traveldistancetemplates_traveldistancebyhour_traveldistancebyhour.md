# Travel Distance per Hour by Type vs Time Template

## Overview and Key Concepts

 

 

This template creates a time plot, with one colored line per object. The line shows the distance traveled for that object from the previous hour, every hour. For more information about Travel templates, see Travel Templates.

 

 

 

## Properties Panels

 

This template uses the following properties panels:

 

- Options
- Settings
- Text
- Colors
- Sorting
- Advanced
