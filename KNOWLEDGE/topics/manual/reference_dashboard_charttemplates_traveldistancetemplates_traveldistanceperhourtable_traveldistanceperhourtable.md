# Travel Distance per Hour Table Template

## Overview and Key Concepts

 

 

This template creates a table chart with two columns. The first column displays the path to the included objects, and the second displays the distance traveled of that object, divided by the total number of hours the model has run so far.

 

For more information about Travel templates, see Travel Templates.

 

 

 

## Properties Panels

 

This template uses the following properties panels:

 

- Options
- Text
- Advanced
