# Milestone Sankey Template

## Overview and Key Concepts

 

 

This template is a Milestone Template. It assumes that you already have a Milestone Collector in your model, and that you want to create a chart based on data from that collector.

 

This template creates a Sankey Diagram, which shows how items flow from one Milestone to the next. Thicker paths indicate that more items flow between two milestones.

 

 

 

## Properties Panels

 

The Milestone Sankey template uses the following properties panels:

 

- Options
- Settings
- Text
- Colors
- Advanced
