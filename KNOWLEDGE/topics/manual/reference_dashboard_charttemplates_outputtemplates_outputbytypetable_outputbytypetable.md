# Output by Type Table Template

## Overview and Key Concepts

 

 

This composite template creates a table chart with two columns. The first column shows the type value. The second column shows the throughput for each type. For more information about composite Output templates, see Output Templates.

 

 

 

## Properties Panels

 

The Output by Type Table template uses the following properties panels:

 

- Options
- Text
- Advanced
