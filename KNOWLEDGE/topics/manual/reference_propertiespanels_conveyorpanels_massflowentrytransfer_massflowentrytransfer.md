# Mass Flow Entry Transfer

The Mass Flow Entry Transfer panel defines properties for a mass flow entry transfer.

 

 

The following properties are on the Mass Flow Entry Transfer panel:

 

### Flow Unit

 

Defines the mass flow unit that will be created from an entering item.

 

### Units per Item

 

Defines the quantity of mass flow units that will be generated from a single entering flow item.
