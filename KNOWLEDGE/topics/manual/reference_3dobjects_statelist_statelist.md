# State List

The following content is a list of state numbers and their respective macros. Whenever you write code that has to do with setting the state of objects, like using the `Object.stop()` command or the utilize task, you can substitute these macros in for the number. Refer to the reference topics for each of the 3D objects for more information about what each state means to each object.
