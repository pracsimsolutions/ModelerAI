# Staytime by Type Table Template

## Overview and Key Concepts

 

 

This composite template creates a table. The table has one row per Type value, and four columns. The first column shows the Type value. The second, third and fourth columns show the Average, Minimum, and Maximum Staytime values, respectively.

 

For more information about composite Staytime templates, see Staytime Templates.

 

 

 

## Properties Panels

 

The Staytime by Type Table template uses the following properties panels:

 

- Options
- Text
- Advanced
