# Label vs Time Template

## Overview and Key Concepts

 

 

This template creates a stair-step chart. Each specified label is represented by a different color, which shows current value over time.

 

For more information about Label templates, see Label Templates.

 

 

 

## Properties Panels

 

The Label vs Time template uses the following properties panels:

 

- Options
- Settings
- Text
- Colors
- Sorting
- Advanced
