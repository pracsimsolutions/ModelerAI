# Push Back Rack

## Overview and Key Concepts

 

The Push Back Rack is a subclass of the Gravity Flow Rack, which is in turn a subclass of the Rack, which is in turn a subclass of the Storage Object. Most logic for the Push Back Rack is implemented as part of its parent classes. The main specialization of the Push Back Rack is that, unlike the Gravity Flow Rack, its pick face is the same side as its "place face." Additionally, placing an item into a slot displaces the other items in the slot, pushing them farther back into the rack.
