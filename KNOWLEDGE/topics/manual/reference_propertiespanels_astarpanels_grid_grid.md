# Grid

The Grid panel defines the height and granularity of the grid.

 

 

The following properties are on the Grid panel:

 

### Node Spacing X

 

This is the distance (in model units) between nodes in the grid in the X direction. Smaller spacing will allow travel members to move more smoothly and direct, however, more nodes will require more processing time to calculate paths.

 

### Node Spacing Y

 

This is the distance (in model units) between nodes in the grid in the Y direction.

 

### Z Position

 

Sets the Z position of this grid.

 

### Set View Grid Z

 

Sets the current Grid Z of the 3D view to the Z position of this grid.
