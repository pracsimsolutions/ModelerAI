# Spinner

The Spinner panel contains options for spinner controls.

 

 

The following properties are on the Spinner panel:

 

### Minimum Value

 

The minimum value the control can be set to.

 

### Maximum Value

 

The maximum value the control can be set to.

 

### Step Size

 

Amount to increase/decrease by.
