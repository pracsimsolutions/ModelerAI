# Route

## Overview and Key Concepts

 

Routes are created by connecting two points together. Task executers travel along routes when executing travel tasks using the GIS Navigator.

 

 

 

 

## States

 

The route does not implement any states.

 

 

 

## Statistics

 

The route does not track any statistics.
