# Node

When a node is highlighted in the tree, the Node panel will appear, displaying rank of the node.

 

 

The following properties are on the Node panel:

 

### Rank

 

This specifies the rank of the highlighted node in its parent's tree. Enter a new rank number or use the up and down arrows to rerank the node.

 

### Data

 

If the node has coupling data, the Data field will display the coupling's associated node. Use the Sampler button to select the node to create a coupling with.

 

### Clear Contents

 

Destroys all subnodes of the highlighted node.

 

### Execute Node

 

Calls nodefunction or executefsnode on the node, executing its associated FlexScript, C++ or DLL code.
