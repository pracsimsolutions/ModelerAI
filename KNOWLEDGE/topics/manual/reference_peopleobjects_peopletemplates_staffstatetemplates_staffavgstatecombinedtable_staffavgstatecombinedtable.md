# Staff Avg State (Combined) Table Template

## Overview and Key Concepts

 

 

This chart template creates a table chart. The template calculates utilization based on time spent in combinations of state and availability status.

 

 

 

## Properties Panels

 

This template uses the following properties panels:

 

- Data
- Options
- Subsets
- Partitions
- Text
- Advanced
