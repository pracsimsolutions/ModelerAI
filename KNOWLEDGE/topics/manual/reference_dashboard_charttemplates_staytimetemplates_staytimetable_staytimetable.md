# Staytime Table

## Overview and Key Concepts

 

 

This template creates a chart with up to four columns. The first column shows the object. The other three columns show the average, minimum, and maximum staytime for the object.

 

For more information about Staytime templates, see Staytime Templates.

 

 

 

## Properties Panels

 

The Staytime Table template uses the following properties panels:

 

- Options
- Text
- Advanced
