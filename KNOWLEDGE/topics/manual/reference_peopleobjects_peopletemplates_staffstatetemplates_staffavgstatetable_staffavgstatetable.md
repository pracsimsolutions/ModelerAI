# Staff Avg State Table Template

## Overview and Key Concepts

 

 

This chart template creates a table chart, and calculates utilization based on each object's time in various states.

 

 

 

## Properties Panels

 

This template uses the following properties panels:

 

- Data
- Options
- Subsets
- Partitions
- Text
- Advanced
