# Composite Output Bar Template

## Overview and Key Concepts

 

 

This template creates a bar chart. There is one bar, and its length is determined by the total throughput (usually output) of all included objects.

 

For more information about Output templates, see Output Templates.

 

 

 

## Properties Panels

 

The Composite Output Bar template uses the following properties panels:

 

- Options
- Settings
- Text
- Colors
- Sorting
- Advanced
