# Travel Distance Table Template

## Overview and Key Concepts

 

 <!-- This section explains what data the chart shows, and any assumptions it makes --> 

This template creates a table chart with two columns. The first column displays the path to the included objects, and the second displays the distance traveled of that object.

 

For more information about Travel templates, see Travel Templates.

 

 

 

## Properties Panels

 

This template uses the following properties panels:

 

- Options
- Text
- Advanced
