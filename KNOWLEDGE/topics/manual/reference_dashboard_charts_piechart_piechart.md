# Pie Chart

## Overview and Key Concepts

 

The Pie chart can be used to visualize and compare similar values from many objects. The example below shows the time in state for several objects.

 

 

There are two possible formats for the data table for the pie chart. The are described in the following sections.

 

### One Pie per Row

 

This is the simpler format. The Pie chart assumes that each row in the table represents a single pie chart. You can specify which columns to draw pie segments for.
