# Composite WIP vs Time Template

## Overview and Key Concepts

 

 

This composite template creates a stair-step chart. The line shows the number of items between the specified objects, over time. For more information about composite WIP templates, see Content Templates.

 

 

## Properties Panels

 

The Composite WIP vs Time template uses the following properties panels:

 

- Options
- Settings
- Text
- Colors
- Sorting
- Advanced
