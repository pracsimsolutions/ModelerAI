# Edit

The Edit panel contains options for edit controls.

 

 

The following properties are on the Edit panel:

 

### Use Minimum Value

 

Set a minimum value for this edit.

 

### Minimum Value

 

The minimum value the control can be set to.

 

### Use Maximum Value

 

Set a maximum value for this edit.

 

### Maximum Value

 

The maximum value the control can be set to.
