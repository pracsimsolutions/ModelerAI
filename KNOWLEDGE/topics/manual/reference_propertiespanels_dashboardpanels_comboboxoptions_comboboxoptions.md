# Combo Box Options

The Combo Box Options panel contains options for combo box and list box controls.

 

 

The following properties are on the Combo Box Options panel:

 

### Options

 

The list of name-value pairs that will be displayed in the combo box list. The name is shown in the combo box's list and the value is the value applied to the control's linked node when selected from the list. Values must be numeric.

  

#### Linking to String Data

 

If a combobox or listbox is linked to a node with string data like a label, selecting an option will set the value of the linked label or node to be the name of the option, rather than its numeric value.
