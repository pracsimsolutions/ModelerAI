# WIP by Type Table Template

## Overview and Key Concepts

 

 <!-- This section explains what data the chart shows, and any assumptions it makes --> 

This composite template creates a table chart with two columns. The first column shows the Type value, and the second shows the Composite WIP, or the number of items between the entry and exit objects of each type. For more information about Composite WIP templates, see Content Templates.

 

 

 

## Properties Panels

 

The WIP by Type Table template uses the following properties panels:

 

- Options
- Text
- Advanced
