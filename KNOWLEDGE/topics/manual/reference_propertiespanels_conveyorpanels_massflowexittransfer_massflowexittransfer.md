# Mass Flow Exit Transfer

The Mass Flow Exit Transfer panel defines properties for a mass flow exit transfer.

 

 

The following properties are on the Mass Flow Exit Transfer panel:

 

### Package Item

 

Defines the flow item that represents one "package" of mass flow units.

 

### Units per Package

 

Defines the quantity of mass flow units that will generate a single package flow item.
