# Content Table Template

## Overview and Key Concepts

 

 <!-- This section explains what data the chart shows, and any assumptions it makes --> 

This template creates a table chart with two columns, Object and WIP. The object column shows the path to the included objects, relative to the model. The WIP column shows the current content (or WIP) of each of the objects.

 

For more information about Content templates, see Content Templates.

 

 

 

## Properties Panels

 

The Content Table Chart uses the following properties panels:

 

- Options
- Text
- Advanced
