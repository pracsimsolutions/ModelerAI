# Capture View

The Capture View panel allows you to save the currently displayed view with a specified resolution.

 

 

The following properties are on the Capture View panel:

 

### Width

 

Specify the desired resolution for the width of the screen capture.

 

### Height

 

Specify the desired resolution for the height of the screen capture.

 

### Capture View

 

Saves the currently displayed view with the specified resolution.
