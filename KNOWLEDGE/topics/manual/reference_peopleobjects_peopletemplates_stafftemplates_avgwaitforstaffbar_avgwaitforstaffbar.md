# Average Wait for Staff Bar People Template

## Overview and Key Concepts

 

 

This template calculates and displays the average time a flowitem (created with the Create Person activity) waits to acquire a Staff Resource.

 

For more information about People Templates, see the People Templates topic.

 

### Subset and Partition Limits

 

This template allows a maximum of two Partitions. If you use any Subsets, then the template will only allow one Partition. If you have two Partitions, the template will not allow you to add any Subsets.

 

 

 

## Properties Panels

 

This template uses the following properties panels:

 

- Data
- Subsets
- Partitions
- Settings
- Text
- Colors
- Sorting
- Advanced
