# Output per Hour Table Template

## Overview and Key Concepts

 

 

This template creates a table chart with two columns. The first column displays the path to the included objects, and the second displays the throughput (usually output) of that object, divided by the total number of hours the model has run so far.

 

For more information about Output templates, see Output Templates.

 

 

 

## Properties Panels

 

The Output per Hour Table template uses the following properties panels:

 

- Options
- Text
- Advanced
