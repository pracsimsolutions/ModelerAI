# State Bar Template

## Overview and Key Concepts

 

 

This chart template creates a bar chart, and calculates utilization based on each object's time in various states. Each segment of the bar chart represents the proportion of time spent in one of the object's states. There is one set of bars per object.

 

 

 

## Properties Panels

 

The State Bar template uses the following properties panels:

 

- Options
- Settings
- Text
- Sorting
- Advanced
