# Average Wait Times Table People Template

## Overview and Key Concepts

 

 

This template calculates and displays average time that flowitems (created with the Create Person activity) have waited for Locations, Staff, Equipment and Transports.

 

For more information about People Templates, see the People Templates topic.

 

### Subset and Partition Limits

 

This templates does not limit the number of Subsets or Partitions.

 

 

 

## Properties Panels

 

This template uses the following properties panels:

 

- Data
- Subsets
- Partitions
- Text
- Advanced
