# Process Flow Object

The Process Flow Object panel displays which Object Process Flows an object is currently attached to.

 

 

The following properties are on the Process Flow Object panel:

 

### Process Flow Class

 

A dropdown of all the process flows the object is attached to.

 

| Button | Explanation |
| --- | --- |
|  | Attach the object to a new Object Process Flow. |
|  | Detach the object from the flow currently selected in the dropdown list. |

 

### Open Process Flow Properties

 

Opens the properties window for the flow currently selected in the dropdown list.

 

### Open Process Flow View

 

Opens a Process Flow view for the flow currently selected in the dropdown list.
