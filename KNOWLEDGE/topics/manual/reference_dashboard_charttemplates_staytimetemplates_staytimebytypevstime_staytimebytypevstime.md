# Staytime by Type vs Time Template

## Overview and Key Concepts

 

 

This template creates a time plot. Each point on the plot represents an item exiting the composite object. The color of each point represents the Type value of the exiting item. The position of the point is determined by when the item exited, and how long the item was within the composite object.

 

For more information about composite Staytime templates, see Staytime Templates.

 

 

 

## Properties Panels
