# Label Table Template

## Overview and Key Concepts

 

 <!-- This section explains what data the chart shows, and any assumptions it makes --> 

This template creates a table chart with two columns, Name and Value. The name column shows the name of the included labels. The Value column shows the current value each of the labels.

 

For more information about Label templates, see Label Templates.

 

 

 

## Properties Panels

 

The Label Table Chart uses the following properties panels:

 

- Options
- Text
- Advanced
