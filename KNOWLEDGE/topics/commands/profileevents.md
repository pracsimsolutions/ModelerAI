# profileevents

```flexscript
profileevents()
```

Prints descriptions of all events currently in the event queue to the output console

Prints to the output console descriptions of all of the events currently waiting in the event queue. The output console must be open to write to it. To open console, use menus View | Output Console.

## Example

```flexscript
profileevents()
```
