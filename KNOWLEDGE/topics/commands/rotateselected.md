# rotateselected

```flexscript
rotateselected(double angle, double x, double y[, obj roottree])
```

Rotates all selected objects in the model around the designated origin x/y location

## Example

```flexscript
rotateselected(90, 5, 0)
This example rotates selected objects 90 degrees around the location (5,0)
```
