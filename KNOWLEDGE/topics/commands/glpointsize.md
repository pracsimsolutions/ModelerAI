# glPointSize

```flexscript
glPointSize(num pointsize)
```

Specifies the point size in pixels for subsequent vertices drawn in OpenGL

Specifies the point size in pixels for subsequent vertices drawn in OpenGL. You can refer to standard OpenGL documentation for more information if needed.

## Example

```flexscript
glPointSize(5);
```
