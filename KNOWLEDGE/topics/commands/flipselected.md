# flipselected

```flexscript
flipselected(num firstX, num firstY, num secondX, num secondY[, node roottree])
```

Flip selected objects across an axis

Flip selected objects across the axis defined by the two passed xy points

## Example

```flexscript
flipselected(1, 5, 2, 7)
This example flips the selected objects across the axis
defined by the points point (1, 5) and (2, 7)
```
