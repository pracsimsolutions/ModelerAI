# programdatadir

```flexscript
programdatadir() -> str
```

Gets the full path to the directory where FlexSim stores program data

Returns a full path to the directory where FlexSim stores program data, such as the help documents.

## Example

```flexscript
programdatadir() returns a string like "C:\ProgramData\Flexsim\Flexsim7\"
```
