# updatereport

```flexscript
updatereport(num/str/node report)
```

For Dynamo use

For Dynamo use. Causes the specified Report to add a new row to its table and populate it correctly. This is used to add entries that are not time-based.

## Example

```flexscript
updatereport("ResultsReport");
```
