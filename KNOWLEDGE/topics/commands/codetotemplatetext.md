# codetotemplatetext

```flexscript
codetotemplatetext(node codeobject, node toobjecttext [, num firstcommentonly])
```

For developer use. Converts C++/Flexscript code into template code. Sets the text in toobjecttext to the template code of the code found in codeobject's text
