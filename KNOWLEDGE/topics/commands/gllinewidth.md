# glLineWidth

```flexscript
glLineWidth(num linewidth)
```

Specifies the line width for subsequent lines draw in OpenGL

Specifies the line width for subsequent lines drawn in OpenGL. You can refer to standard OpenGL documentation for more information if needed.

## Example

```flexscript
glLineWidth(3);
```
