# excelmultitableimport

```flexscript
excelmultitableimport()
```

Starts the automatic importing of data from Excel to FlexSim tables

Starts the automatic importing of data from Excel sheets to FlexSim tables as setup in the Excel Import/Export tool.

## Example

```flexscript
excelmultitableimport()
```
