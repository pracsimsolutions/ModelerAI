# StatisticsCollector

A class that represents a Statistics Collector

## Methods

### getID

```flexscript
double (static) StatisticsCollector.getID(treenode object)
```

Returns a unique ID for a node.

**Parameters:**
- `object` *(treenode)* — The node to get an ID for

**Returns:** The ID of the node

All Statistics Collector IDs are generated as they are needed.
The first time an object is passed to this method, a new ID will be generated.
Subsequent calls to this method for the same treenode will return the same
ID. If the node is temporary, such as a flowitem, token, or any node that
will be destroyed on reset, then the ID will be a incrementing number, starting
with 1. If the node is more permanent, then an ID will be generated by hashing
the full path of the object.

### getPathFromID

```flexscript
string (static) StatisticsCollector.getPathFromID(double id, int maxDepth = 0)
```

Gets the path used to generate an ID for the node.

**Parameters:**
- `id` *(double)* — The ID to retrieve a path for
- `maxDepth` *(int, default 0)* — The number of names to show in the returned path.

**Returns:** The path used to create the given ID.

If the ID is valid, and does not refer to a temporary object,
this method returns the path that was used to create the ID. If maxDepth is
greater than 0, the path will be
shortened, so that only maxDepth names will appear in the path.
If the ID is not valid, or refers to a temporary object, then
an error text will be returned.

### StatisticsCollector

```flexscript
StatisticsCollector (static) StatisticsCollector.StatisticsCollector(Variant id)
```

Creates a reference to a Statistics Collector.

**Parameters:**
- `id` *(Variant)* — The value that specifies a Statistics Collector

**Returns:** A StatisticsCollector reference.

Global Statistics Collectors can be referenced by name or by rank.
A Statistics Collector's node can also be used.

### getAllRowValues

```flexscript
Array StatisticsCollector.getAllRowValues()
```

Gets all row values from the collector.

Returns the row values for all unfinished rows
in the collector. If the collector is set to keep
row values for finished rows, then this method will return
all row values ever used by the collector, except for null
values.

### getRowForValue

```flexscript
int StatisticsCollector.getRowForValue(Variant value)
```

Returns the row number for given value, or 0 if that value is not tied to a row.

**Parameters:**
- `value` *(Variant)* — The value to find a row number for

**Returns:** A row number

This method can be used to determine whether a value
currently has a row in the current collector. For example, if the row mode
is set to Unique Row Values, and you are listening to multiple events,
it could be that you want to allow an event to update the row for the value,
but prevent that value from adding a row. In this case, the event's Condition
might have code like this:
collector.getRowForValue(data.item)

### getValue

```flexscript
Variant StatisticsCollector.getValue(int row, int col, int skipUpdate = 0)
```

Gets a value from the table created by the Statistics Collector.

**Parameters:**
- `row` *(int)* — The row of the desired value
- `col` *(int)* — The column of the desired value
- `skipUpdate` *(int, default 0)* — Choose whether the table should update its values before retrieving the data.

**Returns:** A Variant, the value from the table.

This method is intended to be used within one of the Statistics Collector's
callbacks. In certain cases, it can be desirable to get a value without causing
an update of all Always columns that are still being tracked. This method can
also be used to get a value from the collector's table without wrapping the
node in the Table constructor.
